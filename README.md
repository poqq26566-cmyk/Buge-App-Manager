## Features

### 🚀 Core Capabilities
- **Shizuku Integration** – Operates with Shizuku authorization for elevated control without requiring root.
- **App Management** – Browse and manage both **user-installed** and **system applications** with ease.
- **Material Design 3 UI** – Clean, system-themed interface that follows the latest Android design guidelines.

### 🔐 Permission Manager
- View and modify app permissions by category
- Supported categories include:
  - Microphone
  - Camera
  - Location access
  - And more
- Filter permissions by **granted** or **denied** status

### 🛡️ Privacy & Security
- Quickly identify which apps have access to sensitive permissions
- Optimize your device's security posture with clear permission insights

## How It Works

1. **Authorization** – Uses Shizuku to gain privileged access for app and permission management.
2. **App Overview** – Browse all installed apps in a clean, searchable list.
3. **Permission Management** – Select any app to view and modify its permissions by category.
4. **Privacy Control** – Review granted vs. denied permissions to maintain privacy.

## Requirements

- Android device running **Android 8.0 (API 26) or higher**
- [Shizuku](https://shizuku.rikka.app/) installed and authorized
